# Anil Kumar - Product Manager Portfolio (HTML Version)

This is a standalone HTML/CSS/JavaScript version of the portfolio website.

## Files Included

- `index.html` - Main HTML file with all content
- `style.css` - Combined CSS styles from all components
- `script.js` - JavaScript for interactivity (smooth scrolling, animations)
- `hero-image.png` - Profile/hero image

## How to Use

### Option 1: Open Locally
Simply double-click `index.html` to open it in your default browser.

### Option 2: Use a Local Server
For better performance, use a local server:

```bash
# Using Python 3
python -m http.server 8000

# Using Node.js (if you have http-server installed)
npx http-server

# Using VS Code
# Install "Live Server" extension and right-click index.html > "Open with Live Server"
```

Then navigate to `http://localhost:8000` in your browser.

## Customization

### Update Content
Edit `index.html` to:
- Change text content
- Update project details  
- Modify contact information
- Add/remove sections

### Update Styles
Edit `style.css` to:
- Change colors (search for color variables at top)
- Adjust spacing and layout
- Modify animations

### Update Image
Replace `hero-image.png` with your own image (recommended size: 600x600px or larger)

## Browser Compatibility

Works on all modern browsers:
- Chrome/Edge (latest)
- Firefox (latest)
- Safari (latest)

## Deployment

You can deploy this to any static hosting service:

### GitHub Pages
1. Create a new repository
2. Upload all files
3. Go to Settings > Pages
4. Select branch and save

### Netlify/Vercel
1. Drag and drop the `html-portfolio` folder
2. Your site will be live instantly

### Any Web Host
Upload all files via FTP to your web hosting provider.
